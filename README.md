# Solana Tokens Factory

A DApp for creating and managing tokens on Solana Mainnet.

## Features
- Create tokens with custom names, symbols, and images.
- Bonding curve for buying/selling tokens.
- Automatic listing on Raydium.

## Setup
1. Clone the repository.
2. Open `index.html` in your browser.
3. Connect your Phantom wallet.
4. Create and manage tokens.

## Dependencies
- Solana Web3.js
- Metaplex
- Raydium SDK
